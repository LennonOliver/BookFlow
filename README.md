📚 BookFlow

BookFlow est une application web de gestion de bibliothèque personnelle.  
Elle permet de rechercher des livres, de constituer une collection en ligne et de gérer facilement sa bibliothèque.

Ce projet a été réalisé dans un objectif d’apprentissage et de mise en pratique du développement web front-end.

---

✨ Fonctionnalités

- 🔍 **Recherche de livres** via l’API de [Open Library](https://openlibrary.org/)
- ➕ **Ajout de livres** à une bibliothèque personnelle
- ✍️ **Ajout manuel** de livres lorsque ceux-ci ne sont pas disponibles via l’API
- 📝 **Modification et suppression** des livres de la bibliothèque
- 📄 **Mise en place d’un export de la bibliothèque au format PDF**

---

🛠️ Technologies utilisées

- HTML5  
- CSS3  
- JavaScript (vanilla)  
- API Open Library
